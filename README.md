#### JavaScript 3D library ####

The aim of the project is to create an easy to use, lightweight, 3D library with a default WebGL renderer.

### Usage ###

```
$ npm i
$ npm start
```

### Technologies ###

- three
- Webpack
    - webpack-dev-server
    - webpack-cli
